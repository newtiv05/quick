Test des Traductions

Ouvrez la console du navigateur pour voir les logs de débogage des traductions.

L'application devrait être disponible sur : http://localhost:3000/

Instructions de test :

  1. Ouvrez l'application dans le navigateur
  2. Ouvrez la console du navigateur (F12)
  3. Regardez les logs de débogage des traductions
  4. Changez de langue dans l'interface
  5. Vérifiez que les noms des produits changent
